Result Code 1 is caused by one of the following:
  1. Invalid login information.  Remember, its case-sensitive.
  2. Created separate Payflow Pro password. This is not suggested, it is better to create a separate Payflow Pro User.
  3. IP Restrictions on account.  See Service Settings >> Allowed IP Address in Manager at https://manager.paypal.com.
Result Code 26 is caused by the following:
  1. Invalid login information.  Remember, VENDOR and USER are the same unless you created a separate Payflow Pro User and all fields are case-sensitive.
